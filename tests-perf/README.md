## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/jaktestowac/bughuntfest-playwright-patterns.git
   cd bughuntfest-playwright-patterns/tests-perf
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

## Running Tests

Run all tests:

```bash
npm run test
```
